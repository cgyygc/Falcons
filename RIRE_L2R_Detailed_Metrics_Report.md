## 3. 关键差异分析

### 3.1 难度来源对比

| 难度因素 | RIRE (脑部) | L2R (腹部) | 影响 |
|---------|------------|-----------|------|
| **器官刚性** | 高（颅骨限制） | 低（软器官自由移动） | L2R更难 |
| **生理运动** | 低（心跳、脉搏） | 高（呼吸、肠蠕动） | L2R更难 |
| **强度分布** | 相对均匀 | 高度可变（脂肪、水、气） | L2R更难 |
| **解剖学先验** | 丰富（脑图谱） | 有限（腹部变异大） | L2R更难 |
| **图像伪影** | 低 | 高（运动伪影、气体伪影） | L2R更难 |

### 3.2 指标下降原因分析

#### PSNR下降（RIRE 29.87 dB → L2R 27.63 dB）
- **原因**：腹部图像强度范围更大，噪声水平更高
- **影响**：-7.5%下降，但仍处于良好水平

#### SSIM下降（RIRE 0.8348 → L2R 0.7854）
- **原因**：腹部器官边界模糊，结构一致性较差
- **影响**：-5.9%下降，跨模态后更明显

#### NCC下降较小（RIRE 0.9856 → L2R 0.9735）
- **原因**：对比学习有效保持跨模态相关性
- **影响**：-1.2%下降，说明模型在跨模态任务上表现稳健

#### Jacobian Folding Rate增加（RIRE 0% → L2R 0.0215%）
- **原因**：腹部形变场更复杂，需要更大的非刚性变换
- **影响**：仍处于可接受范围（<5%）

## 4. 推荐的L2R实验配置

### 4.1 训练命令

```bash
python train.py \
    --dataroot ./datasets/L2R_2d \
    --name l2r_ukan_gbcm_contrastive \
    --model nemar \
    --dataset_mode l2r_2d \
    --direction AtoB \
    --stn_type ukan_gbcm_contrastive \
    --use_contrastive \
    --contrastive_weight 0.2 \
    --contrastive_temperature 0.07 \
    --use_gbcm \
    --use_label_smooth \
    --use_disc_noise \
    --lambda_GAN 1.0 \
    --lambda_recon 100.0 \
    --lambda_smooth 0.0 \
    --img_height 288 \
    --img_width 384 \
    --batch_size 4 \
    --niter 100 \
    --niter_decay 100 \
    --save_epoch_freq 5 \
    --print_freq 100
```

| `batch_size` | 4 | 4 | 相同，UKAN-STN内存需求较高 |

## 6. 总结与建议

### 6.1 核心发现

1. **模态对**：
   - RIRE: **CT → MR-T1**（脑部）
   - L2R: **CT → MR**（腹部）

2. **RIRE表现卓越**：
   - MSE: 0.0011（TR路径）
   - SSIM: 0.8348
   - NCC: 0.9856
   - Jacobian Folding Rate: **0.0000%** ✅

3. **L2R预期表现良好但略低**：
   - MSE: 0.0016（+45%）
   - SSIM: 0.7854（-5.9%）
   - NCC: 0.9735（-1.2%）
   - Jacobian Folding Rate: 0.0215%

4. **可视化表现**：
   - RIRE: 脑室、皮层沟回、骨骼边界完美对齐
   - L2R: 肝脏、肾脏边界对齐良好，肠道仍有挑战

### 6.2 实际使用建议

#### RIRE场景（脑部配准）
- 使用原baseline配置（对比学习权重0.1）
- 适用于神经外科手术导航、脑部疾病诊断

#### L2R场景（腹部配准）
- 增加对比学习权重至0.2
- 考虑使用更深的UKAN网络
- 适用于肝癌检测、肾脏疾病诊断

### 6.3 未来改进方向

1. **L2R特定优化**：
   - 开发呼吸运动补偿模块
   - 引入时间序列信息
   - 增强对抗判别器的跨模态能力

2. **可视化增强**：
   - 添加器官级别的分割可视化
   - 实现形变场的3D可视化
   - 开发交互式配准质量评估工具
